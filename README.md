\# zhen



Python + PyQt6 UI + Modbus(可选) + 视频(可选) 项目。



\## Quick Start (Windows)

```powershell

cd C:\\path\\to\\zhen

python -m venv .venv

.\\.venv\\Scripts\\activate

pip install -U pip

pip install -r requirements.txt

python -m app.main


Quick Start (Raspberry Pi OS)
cd /path/to/zhen
python3 -m venv venv
source venv/bin/activate
pip install -U pip
pip install -r requirements.txt
python3 -m app.main
